📊 How These Are Calculated

  • 🎯 Attention (Focus) = Beta / (Alpha + Theta)
    High values indicate alert, processing information. Low values mean relaxed or distracted.
  • 🧘 Meditation (Calm) = Alpha / (Beta + Gamma)
    High values indicate relaxed, wakeful state with quiet mind. Low values mean active thinking or anxiety.
  • 😴 Drowsiness (Sleep Onset) = Delta / (Alpha + Beta)
    High values indicate sleep onset with increased slow-wave activity. Watch for this spike when tired!
